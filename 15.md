# 第十五天

## 目标

掌握窗口的管理

- 左右 - `Ctrl w v`
- 上下 - `Ctrl w s`

### 窗口切换

- `Ctrl w hjkl` - 切换到上 / 下 / 左 / 右方向的窗口
- `Ctrl w w` - 窗口间切换

### 关闭窗口

- `Ctrl w c`- 关闭当前窗口
- `Ctrl w o` - 关闭除当前窗口外的所有窗口

### VS Code

- 新建窗口
  - `Command \`
  - `Command ctrl \`
- 关闭窗口
  - `Command w`
  - `Command k w`
- 窗口切换
  - `Shift + 方向键`  
  - `Shift + ctrl + hjkl`
