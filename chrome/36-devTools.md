### 打开 Devtool

- 打开上次打开的面板
  - `command + option + l`
- 打开控制台
  - `command + option + j`
  - `` ctrl + ` ``
- 打开元素界面
  - `command + shift + c`
  - `command + option + c`
- 清空控制台
  - `command + k`
- 搜索文件
  - `ctrl + p`
- 搜索命令
  - `command + shift + p`
- 切换面板
  - `command + [kkiei`
  - `command + ]`
- 在source panel 中隐藏底部面板
  - `ESC`
