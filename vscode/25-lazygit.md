# 第二十五天

## 目标

搞定常用 lazygit 操作

## 任务点

### 安装

```shell
brew install lazygit
```

### 配置别名

```shell
echo "alias lg = 'lazygit'" >> ~/.zshrc
```

### git add

选中文件列表，按下空格键可以切换暂存状态

### git commit

按下 `c` 键，弹出`commit`框，输入 `commit` 信息，回车键提交

### git discard

按下 `d` 取消修改

### git push

`P` 推送

### git pull

`p` 拉取

### git reset

切换到 commit 列表，选择想要撤销的commit，按下 g 键，弹出选框，选择需要的选项，回车确认
